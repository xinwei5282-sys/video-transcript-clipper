# 隐私说明

本扩展在用户本机运行，用于采集当前打开的视频页面中的媒体地址，并调用用户配置的转写服务。

## 本扩展会读取什么

在用户打开抖音或小红书页面时，本扩展可能读取：

- 当前页面 URL
- 当前页面中的视频或音频资源 URL
- 页面网络请求中的媒体资源 URL
- 转写服务返回的文本结果

## 本扩展会上传什么

取决于用户选择的接口类型：

- 使用阿里百炼 DashScope 时：
  - 小红书：提交视频 URL 给 DashScope Paraformer。
  - 抖音：先在浏览器中下载媒体文件，再上传到用户自己的 DashScope 文件存储，然后调用 Paraformer。
- 使用自定义 API 时：
  - 提交 `videoUrl`、`sourceUrl`、`platform`、`durationSeconds` 给用户配置的 API 地址。

## API Key 存储

用户填写的 API Key 仅保存到本机浏览器的 `chrome.storage.local`。

本项目不内置、不收集、不上传开发者 API Key。

## 日志

扩展会在本机保存最近一次运行日志，用于排查问题。

日志会对常见敏感字段和长 URL 进行脱敏/截断。用户可以在弹窗中清空日志。

## 第三方服务

本扩展可能调用：

- 阿里云 DashScope / 百炼
- 用户自行配置的自定义 API
- 当前页面的视频平台资源地址

请用户自行确认第三方服务的隐私政策和使用条款。

## 数据删除

用户可以通过以下方式删除本地数据：

- 在扩展弹窗点击「清空日志」
- 在扩展设置页清空 API Key 后保存
- 在 Chrome 扩展管理页移除本扩展
