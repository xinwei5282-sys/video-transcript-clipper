视频文案采集器设置

API Key 仅保存在本机浏览器。抖音视频会先上传到用户自己的 DashScope 文件存储后再转写。

自定义厂商模板

适合简单 HTTP 转写接口，不需要服务器。复杂签名、复杂轮询、对象存储中转类厂商应使用内置 provider。

配置教程

平台处理方式：
小红书：通常提交公开视频 URL 给厂商转写。
抖音：优先提取 media-audio，浏览器下载音频后再上传给支持该方式的厂商。

一、阿里百炼 Paraformer
适合：最省心，抖音防盗链场景更稳。
控制台地址：https://bailian.console.aliyun.com/
DashScope 文档：https://help.aliyun.com/zh/model-studio/
需要开通：DashScope API Key、文件上传能力、Paraformer 音频转写。
插件填写：
  接口类型：阿里百炼 Paraformer
  API Key：DashScope API Key
  模型：paraformer-v2
  App ID / API Secret / 自定义 API 地址：不用填
插件调用：
  小红书：直接提交视频 URL 给 DashScope 转写。
  抖音：先下载音频，上传到 DashScope 文件存储，再提交转写。
常见问题：
  401/403：API Key 错误、额度不足、服务未开通。
  文件上传失败：DashScope 文件服务权限或网络问题。

二、火山引擎 / 豆包语音
适合：想测试豆包语音识别大模型，推荐先测短音频。
控制台地址：https://console.volcengine.com/
豆包语音文档：https://www.volcengine.com/docs/6561
极速版 API 文档：https://www.volcengine.com/docs/6561/1631584
进入路径：火山引擎控制台 -> 豆包语音，不是火山方舟。
需要开通：豆包语音 ASR，资源 ID 推荐 volc.bigasr.auc_turbo。
新版控制台填写：
  接口类型：火山引擎/豆包语音
  App ID：留空
  API Key：豆包语音 ASR 的 X-Api-Key
  模型：volc.bigasr.auc_turbo
旧版控制台填写：
  App ID：X-Api-App-Key
  API Key：X-Api-Access-Key
  模型：volc.bigasr.auc_turbo
插件调用：
  小红书：提交公开 URL 到极速版 recognize/flash。
  抖音：下载音频后 base64 上传到极速版 recognize/flash。
限制：
  极速版音频不超过 2 小时、100MB，建议 20MB 内更稳。
常见问题：
  Invalid X-Api-Key：填成了火山方舟 Key，或没有用豆包语音 ASR 的 X-Api-Key。
  资源无权限：没有开通 volc.bigasr.auc_turbo，或模型字段填错。

三、腾讯云 ASR
适合：小红书公开视频 URL；抖音只适合 5MB 内短音频。
控制台地址：https://console.cloud.tencent.com/asr
产品文档：https://cloud.tencent.com/document/product/1093
录音文件识别文档：https://cloud.tencent.com/document/product/1093/37823
进入路径：腾讯云控制台 -> 语音识别 ASR。
需要开通：录音文件识别。
插件填写：
  接口类型：腾讯云 ASR
  API Key：SecretId
  API Secret：SecretKey
  模型：16k_zh
  App ID / 自定义 API 地址：不用填
插件调用：
  小红书：提交公开 URL 创建识别任务，然后轮询结果。
  抖音：下载音频后 base64 上传；超过 5MB 会提示不支持直传。
限制：
  URL 音频需公网可下载；本地音频上传不超过 5MB。
常见问题：
  Failed to download audio file：腾讯云服务器拉不到视频 URL。
  鉴权失败：SecretId/SecretKey 错误，或账号未开通 ASR。

四、其他厂商和自定义模板
OpenAI：浏览器下载后上传，单文件不超过 25MB。
Deepgram / AssemblyAI：当前只提交公开 URL，适合小红书，不适合抖音防盗链 URL。
自定义厂商模板：适合简单 HTTP POST 接口；不支持复杂签名、复杂轮询、回调、对象存储中转。

API Key 只保存在本机浏览器。服务商额度、计费和可用地区以各自控制台为准。
