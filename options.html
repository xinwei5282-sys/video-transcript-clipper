视频文案采集器设置

API Key 仅保存在本机浏览器。抖音视频会先上传到用户自己的 DashScope 文件存储后再转写。

接口格式

阿里百炼模式：
API地址无需填写，插件自动调用：
https://dashscope.aliyuncs.com/api/v1/services/audio/asr/transcription

自定义模式：
POST API地址
Authorization: Bearer API_KEY
Content-Type: application/json

{
  "videoUrl": "...",
  "sourceUrl": "...",
  "platform": "xhs 或 douyin"
}

返回字段支持：content、text、transcript、result。
